# 股票收益率预测模型 (CNN-BiLSTM)

本项目是一个用于量化投资的深度学习回归预测模型。基于混合特征体系（行情特征 + 技术指标 + 情绪特征），利用 CNN-BiLSTM 网络预测多只股票的未来 5 日收益率，并自动推导出预测股价和涨跌方向。

## 🌟 核心特性
- **纯净的样本外预测**：严格按照 `target_date`（目标预测日）划分训练集、验证集和测试集（70/10/20），杜绝未来数据泄露。
- **混合特征输入**：兼容 B 组输出的 25 维日度数值特征表，融合了市场情绪与传统技术指标。
- **自动超参数寻优**：内置网格搜索，自动在验证集上寻找最佳的网络容量与学习率配置。
- **多随机种子验证**：采用 3 个不同随机种子（46, 2024, 3407）重复训练，并输出均值与标准差，确保结果真实可靠。
- **严谨的对照实验**：包含 `CNN-BiLSTM(全特征)`、`CNN-BiLSTM(无情绪)`、`LSTM(单向基线)` 和 `Naive Baseline(朴素基线)`。

## 🚀 快速开始

### 1. 环境准备
推荐使用 Python 3.9+，在终端中运行以下命令安装依赖：
```bash
pip install -r requirements.txt
```

### 2. 数据准备
请确保您的 Excel 数据文件（如 `最终合并数据_给C_全10只.xlsx`）放置在与代码**平级**或**上级**名为 `data` 的文件夹中。
*(注：为保持仓库纯净，数据文件及运行产出已被 `.gitignore` 忽略，需在本地自行准备。)*

### 3. 运行主程序
在根目录下执行：
```bash
python main.py
```
运行结束后，当前目录下会自动生成：
- `outputs/`：包含模型评估指标（均值/标准差）、超参数记录，以及供 D 组回测的信号表 `predictions_for_D_test.csv`。
- `figures/`：包含训练 Loss 曲线、预测走势对比图、模型性能柱状图。
- `models/`：包含保存的 `.keras` 格式权重文件。

### 4. 重新生成图表 (可选)
如果您修改了绘图逻辑，无需重新训练模型，直接运行：
```bash
python regenerate_figures.py
```

## 📊 实验结论摘要
在最新的 10 只股票测试集联合评估中：
- 引入情绪特征的 **CNN-BiLSTM 实验组** 获得了最高的 **方向准确率 (55.26%)** 和最小的价格 RMSE (25.36)。
- 相较于无情绪对照组 (48.21%) 和朴素基线 (45.19%)，实验组在预测稳定性和收益率误差控制上表现出显著的结构性优势。
